# java-opengl-2048

## Build instructions

### Init

`gradle init`

### Refresh dependencies

`./gradlew build --refresh-dependencies`

### Run

Linux/Mac:
`./gradlew run`
Windows:
`.\gradlew run`

### Git pull

`git pull`

### Git commit & push

`git add <file>` where `<file>` can be a path to a file or folder or `*` (all files) or `-a` (edited files)
`git commit -m "<message>"`
`git push -u origin main`
